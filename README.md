# imagemin

Use Imagemin to compress images

### Install dependencies:

```bash
npm install
```

### Lossy:

```bash
npm run lossy
```

### Lossless:

```bash
npm run lossless
```

[Readme](https://milanaryal.com.np/how-to-compress-images-using-imagemin/).
